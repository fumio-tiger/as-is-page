# Screen to PDF (A4) - Chrome拡張機能

画面の見た目のままA4サイズでPDF印刷するためのChrome拡張機能です。特にO'Reilly学習サイトでの技術書の印刷に最適化されています。

## 機能

- 🖨️ **画面の見た目のまま印刷**: ブラウザの表示をそのままA4サイズでPDF化
- 📄 **A4サイズ対応**: 20mmマージンでA4サイズに最適化
- 🎯 **O'Reilly学習サイト専用最適化**: レイアウト崩れを防ぐ特別な処理
- ⌨️ **ショートカットキー対応**: `Ctrl+Shift+Y`（Mac: `Command+Shift+Y`）で簡単実行
- 🔧 **自動レイアウト調整**: 右余白の原因となる要素を自動で調整

## インストール方法

### 1. ファイルのダウンロード
このリポジトリをクローンまたはZIPファイルをダウンロードしてください。

### 2. Chrome拡張機能として読み込み
1. Chromeブラウザを開く
2. アドレスバーに `chrome://extensions/` と入力
3. 右上の「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. このプロジェクトのフォルダを選択

### 3. 使用方法
- **方法1**: 拡張機能のアイコンをクリック
- **方法2**: ショートカットキー `Ctrl+Shift+Y`（Mac: `Command+Shift+Y`）を押す

## 対応サイト

### 全般的な対応
- 一般的なWebサイト
- ブログ記事
- 技術文書サイト

### 特別最適化済み
- **O'Reilly学習サイト** (`oreilly.com`, `learning.oreilly.com`)

  - 右余白の完全除去

## 技術仕様

### ファイル構成
```
as-is-page/
├── manifest.json          # 拡張機能の設定ファイル
├── background.js          # バックグラウンドスクリプト
├── print-screen-pdf.js    # メインの印刷処理スクリプト
└── README.md             # このファイル
```

### 主要機能
- **レイアウト最適化**: Grid/Flexレイアウトの1カラム化
- **要素の非表示**: サイドバー、広告、目次などの不要要素を非表示
- **LazyLoad対策**: 遅延読み込みコンテンツの完全表示
- **エラーハンドリング**: 印刷失敗時の適切なエラーメッセージ

## トラブルシューティング

### 印刷がうまくいかない場合
1. ページの読み込みが完了していることを確認
2. ブラウザのコンソール（F12）でエラーメッセージを確認
3. ページを一度リロードしてから再試行

### レイアウトが崩れる場合
- 一部の特殊なサイトでは、CSSの競合によりレイアウトが崩れる可能性があります
- その場合は、ブラウザの標準印刷機能をご利用ください
